# Blog Images Directory

This directory contains images for blog articles.

## Image Requirements

- **Featured Images**: 1200x630px (optimal for social sharing)
- **In-article Images**: 800x600px or larger
- **Format**: WebP, JPEG, or PNG
- **Optimization**: Compress images for web delivery

## Placeholder Images

Current blog articles use placeholder image URLs. Replace these with actual images:

1. `best-vapes-2025.jpg` - Hero image of various vape devices
2. `nicotine-guide.jpg` - Infographic about nicotine strengths
3. `crystal-bar-guide.jpg` - Crystal Bar product lineup
4. `refillable-vapes.jpg` - Refillable pod systems comparison

## Recommended Stock Photo Sources

- Unsplash.com
- Pexels.com
- Custom product photography from Shopify

## Adding New Images

1. Place images in this directory
2. Update the `featuredImage` field in the blog article
3. Ensure images are optimized (< 200KB for featured images)
